Flask application with MongoDB database containg airbnb sample dataset to be queried via REST api endpoints for CSCI-621 Term paper project.

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Install Docker.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/anuraj3095/airbnb_backend.git
   ```
2. Run docker-compose up
   ```sh
   npm install
   ```
